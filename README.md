<summary><h2>⚙️ Setup Guide — How to Enable Every Widget</h2></summary>
<br/>

**1. Repo name matters.** Create a repo named **exactly** `dharmik-rabari-03` (same as your username) and check "Add a README file" — GitHub then shows it on your profile automatically.

**2. Snake contribution animation**
- In your special repo, go to `Settings → Actions → General → Workflow permissions` and enable **Read and write permissions**.
- Add `.github/workflows/snake.yml` with the official [Platane/snk](https://github.com/Platane/snk) action (schedule + push trigger). It generates the SVG to an `output` branch automatically — the URL in this README already points there.

**3. GitHub Action workflows (general)**
- Any automation (snake, Spotify, stats refresh) lives under `.github/workflows/*.yml` in your profile repo. Commit the workflow file, then check the **Actions** tab to confirm it runs.

**4. Activity Graph / GitHub Stats / Streak Stats**
- These are hosted APIs (`github-readme-stats`, `github-readme-streak-stats`, `github-readme-activity-graph`) — no setup needed, just swap `username=dharmik-rabari-03` if you ever rename. If images look stale, hosted instances sometimes rate-limit; you can self-host any of them by forking the repo and deploying to Vercel.

**5. Visitor Counter**
- Already active via `komarev.com/ghpvc` — it counts automatically the first time the README is viewed publicly. No token required.

**6. GitHub Trophies**
- Already active via `github-profile-trophy.vercel.app` — updates automatically as your account grows (commits, PRs, stars, followers, etc).

**7. Profile Summary Cards**
- Already active via `github-profile-summary-cards.vercel.app`. Optional: fork that repo + deploy your own Vercel instance for faster/more reliable loads.

**8. Typing SVG**
- Already active via `readme-typing-svg.demolab.com`. Customize text by editing the `lines=` parameter in the URL (`;`-separated phrases).

**9. 3D Contribution Calendar (optional, advanced)**
- GitHub doesn't natively support 3D — but you can generate one with [`yoshi389111/github-profile-3d-contrib`](https://github.com/yoshi389111/github-profile-3d-contrib) via a scheduled GitHub Action, which commits a 3D SVG image you then embed like the snake animation.

**10. Spotify Now Playing**
- Fork [`kittinan/spotify-github-profile`](https://github.com/kittinan/spotify-github-profile), follow its OAuth setup to link your Spotify account, deploy it (Vercel), then replace the placeholder image URL in the Spotify section above with your generated one.

**11. WakaTime Coding Stats (bonus)**
- Install the WakaTime extension in your editor, then add the [`athul/waka-readme`](https://github.com/athul/waka-readme) GitHub Action to auto-post your weekly coding stats into this README between two `<!--START_SECTION:waka-->` / `<!--END_SECTION:waka-->` comment markers.
